# 🤖 Projeto de Inteligência Artificial

## 📌 Descrição

Projeto desenvolvido para análise de dados e aplicação de conceitos de inteligência artificial utilizando Python.

O objetivo é analisar padrões, gerar previsões e automatizar processos através de algoritmos inteligentes.

---

## 🚀 Tecnologias Utilizadas

- Python
- Pandas
- Scikit-Learn
- Jupyter Notebook
- GitHub

---

## 📊 Funcionalidades

- Análise de dados
- Treinamento de modelos
- Geração de gráficos
- Predição de resultados

---

## 🧠 Aprendizados

Durante o desenvolvimento aprendi:

- Manipulação de dados
- Machine Learning
- Análise estatística
- Visualização de dados

---

pip install pandas scikit-learn matplotlib
